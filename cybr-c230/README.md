**Term:** Spring 2026
**Instructor:** Professor Tobi West
**Certification Alignment:** CompTIA Security+ SY0-701

## Course Overview

Hands-on network security labs aligned with CompTIA Security+ domains
covering malware analysis, social engineering, cryptography, PKI,
web application attacks, and mobile device security.
